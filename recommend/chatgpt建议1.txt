# 致命问题1

K线绘制存在BUG

你这里：

```python
y_op = _scale_to_img(np.array([op]))
y_cl = _scale_to_img(np.array([cl]))
```

注意：

这里只传入一个数。

---

_scale_to_img()

内部：

```python
v_min = v.max()
v_max = v.min()
```

只有一个值：

```python
v_min == v_max
```

于是：

```python
return IMG_SIZE // 2
```

结果：

```python
open
close
```

永远在中间。

---

也就是说：

你的蜡烛实体基本失效。

这是严重BUG。

---

应该：

统一缩放：

```python
all_price = np.concatenate([
    open_,
    high_,
    low_,
    close_
])
```

得到一个scale函数。

然后：

```python
open_y
close_y
high_y
low_y
```

全部共用。
